## Описание программы
Random Task Generator — это цифровой помощник, который выбирает одну случайную задачу из вашего личного списка дел.Программа Выбирает случайное дело из списка, перемешивает рутинные задачи, выдает случайные идеи или темы.
---

- Библиотеки: pip install matplotlib requests
