## Описание программы
Book Tracker — приложение для хранения списка прочитанных книг.

Возможности:
- Добавление книг
- Просмотр списка
- Фильтрация по жанру
- Фильтрация по количеству страниц
- Сохранение данных в JSON
- Автоматическая загрузка данных
---
## Требования для запуска
Что нужно установить на компьютер:
- Python 3.10 или выше
- Библиотеки: pip install matplotlib requests
---

```bash
git clone https://github.com/твой-логин/Book Tracker.git
